# Card background images

Drop monthly background images here, named exactly:

`january-1.png`  `january-2.png`  `february-1.png` … `december-2.png`

- `-1` = the **Soft (watercolor)** style, `-2` = the **Dreamy (photo)** style.
- .png, .jpg, .jpeg, or .webp all work.
- Wide landscape (1536x1024). Missing months fall back to the Classic gradient automatically.
